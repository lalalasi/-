小皇后接口文档

备注：所有接口请求方式：get ，返数据回格式：json

数据返回格式

{
"status": 1, // 1为成功，0为失败，失败时message返回错误提示
"message": "",
"data": {}
}

说明：

  1. 页面所有数据都为后台动态管理。需要从接口读取
  2. 导航的 会员中心、场地与设施、赞助与合作、关于我们 栏目下的二级分类动态管理，id不固定，需要动态接口读取
  3. 导航的 演出购票、资讯中心 栏目id后台已固定，分类名称动态管理
